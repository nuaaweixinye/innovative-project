1. 核心操作
操作	说明
无人机巡逻	程序启动后自动沿fence_path.csv航点循环巡逻
选择跟踪目标	在检测窗口中用鼠标左键点击目标（人 / 车 / 卡车），无人机将暂停巡逻并悬停跟踪该目标
取消跟踪	再次点击检测窗口，无人机恢复巡逻
退出程序	按ESC键关闭检测窗口，或在终端按Ctrl+C

2. 界面说明
检测窗口（标题：YOLO + DeepSORT 目标跟踪）中元素：
绿色细框：普通目标（未跟踪）
绿色粗框：选中的跟踪目标，显示跟踪目标 ID X / 距离: Xm
终端输出：实时显示检测到的目标数量（如人：2个，汽车：1个）

有两个窗口，一个opencv，一个yolo本身的

扩展说明
    1. 视频保存
    检测过程会自动保存为 MP4 文件，路径：output_video/airsim无人机_驻地边界.mp4，可修改代码中output_path变量调整保存路径。
    2. 参数调整
    巡逻速度：修改velocity变量（默认 2m/s）
    安全高度：修改safety_margin变量（默认 20m，无人机将高于航点该高度飞行）
    跟踪类别：修改track_classes变量（如添加3表示摩托车，参考 COCO 类别 ID）

# 无人机智能控制系统项目说明

## 1. 项目概述
本项目是一个基于AirSim仿真环境的多无人机智能控制系统，提供了图形用户界面用于任务生成、无人机控制和实时操控功能。

## 2. 主要功能模块

### 2.1 任务管理模块
- 支持简单任务和复杂任务的生成
- 任务参数包括成本(Cost)和执行时间(Time)
- 任务分配系统可将任务分配给特定无人机或所有无人机

### 2.2 无人机控制模块
- 支持多无人机同时控制
- 提供两种控制方式：
  * 移动到指定位置：设置X、Y、Z坐标
  * 按速度移动指定时间：设置X、Y、Z方向的速度和持续时间

### 2.3 实时控制模块
- 键盘控制功能，支持W/S/A/D上下左右移动
- 方向键控制上升/下降/旋转
- 速度可调节
- 支持起飞/降落功能
- 可选择控制特定无人机

## 3. 文件结构
- `main.py`: 主程序入口，包含GUI界面
- `config.py`: 配置文件，定义无人机数量、最大资源等参数
- `Drone.py`: 无人机类，处理单个无人机的任务执行和资源管理
- `Contoller.py`: 控制器类，管理多个无人机和任务调度
- `Task.py`: 任务管理类，处理简单任务和复杂任务
- `SingleDroneControl.py`: 单无人机实时控制功能

## 4. 使用说明

### 4.1 任务生成页面
- 选择任务类型：简单任务或复杂任务
- 简单任务：输入Cost和Time参数
- 复杂任务：输入任务列表，格式如[(1,2), [(2,2), (30,8)]]
- 点击"生成任务"创建任务
- 点击"分配任务"将任务分配给无人机

### 4.2 无人机控制页面
- 选择要控制的无人机（单个或全部）
- 选择控制方式（移动到位置或按速度移动）
- 输入相应参数
- 点击"执行移动"控制无人机

### 4.3 实时控制页面
- 点击"开始实时控制"打开控制窗口
- 设置移动速度
- 选择要控制的无人机
- 点击"起飞"使无人机起飞
- 使用键盘进行实时控制：
  * W/S: 前进/后退
  * A/D: 左移/右移
  * ↑/↓: 上升/下降
  * ←/→: 左转/右转
- 点击"降落"使无人机降落
- 点击"退出控制"结束实时控制

## 5. 技术特点
- 基于AirSim仿真环境
- 面向对象设计
- 图形用户界面
- 任务队列管理
- 资源管理机制
- 实时控制功能

## 6. 注意事项
- 运行前确保AirSim仿真环境已启动
- 无人机需要先起飞才能执行移动命令
- Cost参数不能超过100
- 所有输出信息将在终端打印
